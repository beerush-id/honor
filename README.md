# @beerush/honor

Honor is a REST API framework built on top of Hono.

## ğŸ  Getting Started

To start using Honor quickly, you can clone the starter package:

```bash
git clone git@github.com:beerush-id/honor-starter.git
```

Install dependencies:

```bash
bun install
```

or

```bass
npm install
```

or

```bash
yarn install
```

To run:

```bash
bun dev
```
